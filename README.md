크리스마스와 함께 찰칵!

여러 재미있는 스티커를 사진에 자동으로 붙여주는 프로그램입니다. 사용자의 성별을 인식하여 각기 다른 스티커를 붙여주기도 하여, 단체 사진에 사용하면 재미있는 스티커 사진을 만들 수 있습니다. 

사용한 패키지
cv2, dlib, numpy

실행방법
1. 사용자의 사진이 저장된 경로를 입력합니다.
2. 이후 프로그램이 조건에 맞는 스티커를 사진에 붙여줍니다.

최대한 이목구비가 잘 드러나는 사진일수록 스티커가 정확하게 붙습니다.
